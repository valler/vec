Composition

Every vector can be expressed as a linear combination of a basis. The standard basis for a space is given by basis. It is constant for the modules with a predefined space. In general you have to provide the number of dimensions. The function vec creates a vector. The function add performs vector addition. The function sum is the varyadic equivalent. The function sMul implements scalar multiplication. Use the function mul for componentwise multiplication. The function pro is the varyadic version of mul. The vector one is the sum of the standard basis. The function sAdd adds a scalar multiple of the vector one. The function sca creates a scalar multiple of the vector one. The vector zero is a special case of that and equals scalar multiplication by zero. The functions pow and sPow generalize products of equal terms. The functions mad and sMad perform addition after multiplication.
