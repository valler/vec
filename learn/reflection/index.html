Reflection

This is about reflection in a broader sense, not only through hyperplanes. The function refl returns the mirror image through the hyperplane. The plane is represented by a normal vector. Reflection through a line can be done with the function conj. The line is represented by a unit vector. The name stands for conjugate, not to be confused with the complex conjugate. Point reflection around the origin is simply negation, for example with the function neg.
