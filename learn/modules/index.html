Modules

Focus on four modules: vec, vec2, vec3, and vec4. Start at the root of the repo. Enter the lib. Choose between Deno, TypeScript, and JavaScript. The module vec is the most general. It has no upper limit on the number of dimensions. The others represent different subspaces. The interfaces are similar, but differ where it makes sense.

Note: Module syntax varies across environments. The examples use bare paths as an abstraction.
