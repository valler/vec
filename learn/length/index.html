Length

The length of a vector is defined as the square root of the dot product with itself. The dot product of a vector with itself equals the squared length of a vector. The function dot computes the dot product. The function len returns the length. The function sqrLen returns the squared length. The function dist computes the distance between two points represented by position vectors. The function sqrDist returns the distance squared.
