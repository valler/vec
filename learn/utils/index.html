Fetching text

The function fetchShader is a thin wrapper around fetch. The chosen interface suites fetching shaders, because they often come in pairs. It simply fetches text.

Casting CSS color functions

The function vecFromCSSRGBA takes a CSS rgb or rgba string as an argument. It returns an Array of three or four floats.
