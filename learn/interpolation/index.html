Interpolation

The function lerp returns a linear interpolation between two vectors. The function mix interpolates componentwise. The function slerp performs spherical linear interpolation. It is not defined for collinear vectors. It reduces to linear interpolation in the limit approaching collinearity. This library does not implement a function which treats collinear vectors as if they were not. The limit fallback would be just one of several solutions for this edge case, depending on the situation.
